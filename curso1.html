Curso Introductorio de Power BI

Descripción del curso:

Este curso es ideal para aquellos que desean aprender los conceptos básicos de Power BI desde cero. Aprenderás a conectar datos, crear informes interactivos y cuadros de mando, y utilizar herramientas como Power Query y DAX para transformar y analizar datos.

Contenido del curso:

  • Introducción a Power BI Desktop y Power BI Pro
  • Conexión de datos desde Excel, páginas web y más
  • Creación de gráficos y mapas interactivos
  • Uso de Power Query para la extracción y transformación de datos
  • Desarrollo de informes y paneles interactivos
  • Suscripción a informes y visualizaciones

Requisitos:

No se requiere experiencia previa. Solo necesitas una computadora con Windows 10 y acceso a Internet.
